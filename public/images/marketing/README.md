# Marketing Images Directory

This directory contains branded merchandise and product images for the marketing site.

## Directory Structure

- `hero/` - Hero section images
  - `product-showcase.webp` - Main product showcase for homepage hero
  - `buyer-hero.webp` - Hero image for For Buyers page
  - `supplier-hero.webp` - Hero image for For Suppliers page

- `products/` - Product showcase images
  - Branded merchandise images (boxes, t-shirts, mugs, etc.)
  - Should feature HOLESCALE. or HS. branding
  - Clean, minimalist product photography
  - Light backgrounds matching the cardboard texture aesthetic

- `features/` - Feature section images
  - `create-rfq.webp` - Image for "Post Your Request" step
  - `compare-quotes.webp` - Image for "Compare Quotes" step
  - `order-tracking.webp` - Image for "Order & Track" step

## Image Requirements

- Format: WebP preferred, fallback to JPG/PNG
- Style: Clean, minimalist product photography
- Background: Light backgrounds matching cardboard texture aesthetic
- Branding: HOLESCALE. or HS. branding visible where appropriate
- Quality: Professional studio lighting, high resolution
- Optimization: Images should be optimized for web (compressed but high quality)

## Image Naming Convention

- Use kebab-case (lowercase with hyphens)
- Include descriptive names (e.g., `product-showcase.webp`)
- Include format extension (.webp, .jpg, .png)
